
👻 GhostLink | Professional P2P Chat
GhostLink is a secure, ephemeral, peer-to-peer (P2P) messaging application that runs entirely in your browser. Built with a focus on privacy and modern design, it requires no server-side database—messages flow directly between users.
Featuring the stunning "Deep Aurora" UI theme, GhostLink offers a premium experience on both desktop and mobile devices.
✨ Key Features
 * 🔒 Zero-Server Storage: Messages are peer-to-peer. Once the tab is closed, data is gone forever.
 * ⚡ Instant Connection: Auto-generates a unique 4-digit ID for quick pairing.
 * 📱 Mobile-First Design: Fully responsive layout with a slide-out "Control Center" sidebar for mobile users.
 * 🎨 Aurora UI: A deep, dark aesthetic with glassmorphism effects, ambient background animations, and smooth transitions.
 * 📂 Rich Media Sharing:
   * Send Photos (with preview).
   * Send Videos (with in-chat player).
   * Share Code Snippets (with auto-syntax highlighting via Highlight.js).
 * 💎 Open Source: Transparent code base for trust and community contribution.
🛠️ Tech Stack
 * Frontend: HTML5, CSS3 (Custom Properties & Animations), Vanilla JavaScript (ES6+).
 * P2P Networking: PeerJS (WebRTC wrapper).
 * Icons: FontAwesome 6.
 * Syntax Highlighting: Highlight.js.
🚀 Getting Started
Since GhostLink is built as a Single Page Application (SPA) contained within one file, deployment is incredibly simple.
Option 1: Run Locally
 * Download the index.html file.
 * Open it directly in any modern web browser (Chrome, Firefox, Edge, Safari).
 * Note: For P2P connections to work across different networks, you must have an internet connection (to reach the STUN servers).
Option 2: Deploy to Web
You can host this immediately on GitHub Pages, Netlify, or Vercel.
 * Push the index.html to a GitHub repository.
 * Enable GitHub Pages in settings.
 * Your secure chat is now live!
📖 Usage Guide
 * Start the App: Open the link. You will be assigned a random 4-digit Secure ID (visible in the sidebar).
 * Share ID: Copy your ID and send it to your peer via another channel.
 * Connect:
   * Open the Sidebar (Hamburger Menu).
   * Enter your friend's 4-digit ID in the "Connect to Peer" box.
   * Click Connect.
 * Chat: Once the status dot turns green, you can start messaging.
 * Share Media: Click the + (Plus) icon next to the chat bar to send images, videos, or code snippets.
